# div-mod

Дается два числа a и b. Напишите функцию которая возвращает результат деления и остаток по модулю

_Дополните функцию divAndMod что бы решить задачу_

Пример для ```` a = 8, b = 5 ````

```
8 / 5 = 1
8 % 5 = 3
```

## Ограничения
* ``` 0 < a < 10_000 ```
* ``` 0 < b < 10_000 ```

### Sample Input
5 2 

### Sample Output
2 1

# currency-exchange

Вам нужно написать функцию по переводу долларов в тенге.

Дается _a_ долларов, которые нужно перевести по курсу _b_ тенге за доллар.

_Дополните функцию calculateExchange что бы решить задачу_

## Ограничения
* ``` 0 < a <= 1_000_00 ```
* ``` 0 < b <= 2_147 ```

### Sample Input
1 430

### Sample Output
430

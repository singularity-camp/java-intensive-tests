# prev-next
Дается число a, написать функцию которая возвращает предыдущее, текущее и следующее числа в формате указанном ниже.

_Дополните функцию prevNext что бы решить задачу_

Формат для ``` a = 2022 ```
``` 2021 -> 2022 -> 2023 ```

## Ограничения 
* ``` -2,147,483,648 < a < 2,147,483,647 ```

### Sample Input
2022

### Sample Output
2021 -> 2022 -> 2023

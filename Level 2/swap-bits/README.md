# swap-bits

Реализуйте метод, который меняет местами первые 4 бита с остальными 4 и возвращает результат.

_Дополните функцию swap в классе SwapBits что бы решить задачу_

## Примеры
* ``` 0000 1111 -> 1111 0000 ```
* ``` 0110 0111 -> 0111 0110 ```

## Ограничения
* ``` 0 <= a <= 255 ```

### Sample Input
15

### Sample Output
240
